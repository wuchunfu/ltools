# 🤖 AI 翻译

基于 AI API 的智能翻译工具，支持多种 AI 服务。

## 功能特性

- ✅ 多 AI 服务支持
- ✅ 云端 API（OpenAI、Anthropic、DeepSeek）
- ✅ 本地 Ollama 支持
- ✅ 多语言互译
- ✅ 快速响应
- ✅ 上下文理解

## 支持的 AI 服务

| 服务 | 类型 | 特点 |
|------|------|------|
| OpenAI | 云端 API | GPT-3.5/4，高质量翻译 |
| Anthropic | 云端 API | Claude，自然流畅 |
| DeepSeek | 云端 API | 国产大模型，中文优化 |
| Ollama | 本地部署 | 完全离线，隐私保护 |

> **注意**：这不是离线翻译，需要配置 AI 服务。

## 使用方法

### 配置 AI 服务

**方式 1：云端 API**

1. 选择 AI 服务商
2. 获取 API Key
3. 在设置中配置 API Key
4. 选择模型
5. 测试连接

**方式 2：本地 Ollama**

1. 安装 Ollama
2. 下载翻译模型
3. 启动 Ollama 服务
4. 在设置中配置 Ollama 地址
5. 选择模型

### 翻译文本

1. 打开全局搜索 (`Cmd/Ctrl+5`)
2. 输入 `translate` 或 `翻译`
3. 输入或粘贴文本
4. 选择目标语言
5. 查看翻译结果

### 语言选择

支持的语言：
- 中文（简体/繁体）
- 英语
- 日语
- 韩语
- 法语
- 德语

- 俄语

### 快速翻译

- 自动检测源语言

## 翻译模式

## API 配置

### OpenAI

```yaml
服务商: OpenAI
API Key: sk-xxx
模型: gpt-3.5-turbo
温度: 0.3
```

**获取 API Key**：
1. 访问 https://platform.openai.com/api-keys
2. 创建新的 API Key
3. 复制并保存

### Anthropic

```yaml
服务商: Anthropic
API Key: sk-ant-xxx
模型: claude-3-sonnet-20240229
温度: 0.3
```

**获取 API Key**：
1. 访问 https://console.anthropic.com/
2. 创建 API Key
3. 复制并保存

### DeepSeek

```yaml
服务商: DeepSeek
API Key: sk-xxx
模型: deepseek-chat
温度: 0.3
```

**获取 API Key**：
1. 访问 https://platform.deepseek.com/
2. 创建 API Key
3. 复制并保存

### Ollama 本地

```yaml
服务商: Ollama
地址: http://localhost:11434
模型: llama2
温度: 0.3
```

**安装 Ollama**：
```bash
# macOS/Linux
curl -fsSL https://ollama.com/install.sh | sh

# 下载模型
ollama pull llama2
ollama pull qwen2
```

## 隐私保护

### 云端 API

- 数据通过 HTTPS 加密传输
- API 提供商可能存储数据
- 遵守各服务商隐私政策

### 本地 Ollama

- 完全本地处理
- 不上传任何数据
- 隐私保护优先

### 建议

- 敏感信息使用本地 Ollama
- 普通翻译可使用云端 API
- 注意保护 API Key

## 常见问题

### Q: API 调用失败？
**A**:
1. 检查 API Key 是否正确
2. 检查网络连接
3. 检查账户余额
4. 查看服务商状态页

### Q: 翻译速度慢？
**A**:
1. 检查网络连接
2. 尝试其他模型
3. 减少文本长度
4. 使用更快的模型

### Q: 翻译质量差？
**A**:
1. 尝试更强大的模型
2. 提供更多上下文
3. 使用专业领域模型
4. 手动校对结果

### Q: Ollama 连接失败？
**A**:
1. 确认 Ollama 服务已启动
2. 检查端口是否正确
3. 确认模型已下载
4. 查看服务日志

### Q: 费用太高？
**A**:
1. 使用本地 Ollama（免费）
2. 选择更便宜的模型
3. 优化提示词长度
4. 使用缓存减少重复翻译
