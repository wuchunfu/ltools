# 插件概览

LTools 采用插件化架构，所有功能都以插件形式提供。你可以根据需要启用或禁用插件。

## 插件分类

### 🎵 媒体工具

| 插件 | 描述 | 关键词 |
|------|------|--------|
| [音乐播放器](./music-player) | 内置音源服务，支持多音源、歌词同步、播放列表 | music, 音乐 |
| [截图工具](./screenshot) | 截图、标注、贴图，微信风格编辑器 | screenshot, 截图 |
| 图床 | 图片上传和管理 | image, 图床 |

### 🛠️ 开发工具

| 插件 | 描述 | 关键词 |
|------|------|--------|
| [JSON 编辑器](./json-editor) | 格式化、验证和编辑 JSON 数据 | json, JSON |
| Hosts 管理 | 编辑和管理系统 hosts 文件 | hosts |
| 隧道管理 | SSH 隧道管理工具 | tunnel, 隧道 |
| 书签搜索 | 搜索 Chrome/Firefox/Safari 浏览器书签 | bookmark, 书签 |

### ⚡ 效率工具

| 插件 | 描述 | 关键词 |
|------|------|--------|

| 看板 | 任务看板管理 | kanban, 看板 |

### 💻 系统工具

## 插件管理

### 启用/禁用插件

1. 打开「设置」→「插件」
2. 找到要管理的插件
3. 切换开关启用或禁用

### 查看插件详情

点击插件卡片可以查看：
- 插件描述和版本
- 所需权限
- 使用统计
- 快捷键设置

## 插件权限

插件可能需要以下权限：

| 权限 | 说明 |
|------|------|
| 文件系统 | 访问文件和目录 |
| 网络 | 发起网络请求 |
| 剪贴板 | 读写剪贴板内容 |
| 通知 | 显示系统通知 |
| 进程 | 管理系统进程 |

首次使用需要权限的插件时，会弹出权限请求对话框。
