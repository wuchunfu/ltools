# 插件管理

管理 LTools 插件的启用、禁用和配置。

## 功能特性

- ✅ 查看所有插件
- ✅ 启用/禁用插件
- ✅ 查看插件详情
- ✅ 管理插件权限
- ✅ 插件使用统计

## 打开插件管理

### 方式 1：从设置打开

1. 打开「设置」
2. 选择「插件」标签
3. 查看和管理插件

### 方式 2：从主界面打开

1. 在主界面点击「插件市场」
2. 浏览和管理插件

## 插件列表

### 查看插件

插件列表显示：
- 插件图标
- 插件名称
- 插件描述
- 启用状态

### 插件分类

插件按类别分组：
- 媒体工具
- 开发工具
- 效率工具
- 系统工具
- 实用工具

## 启用/禁用插件

### 启用插件

1. 找到要启用的插件
2. 点击开关切换为启用状态
3. 插件立即可用

### 禁用插件

1. 找到要禁用的插件
2. 点击开关切换为禁用状态
3. 插件功能停止

> **注意**：禁用插件不会删除插件数据，重新启用后恢复。

## 插件详情

### 查看详情

点击插件卡片查看详细信息：

**基本信息**：
- 插件名称和版本
- 作者信息
- 描述说明
- 插件类型

**权限要求**：
- 所需权限列表
- 权限说明
- 授权状态

**使用统计**：
- 使用次数
- 最后使用时间
- 使用频率

**快捷键**：
- 已设置的快捷键
- 快捷键修改入口

## 插件权限

### 权限类型

| 权限 | 说明 |
|------|------|
| 文件系统 | 访问文件和目录 |
| 网络 | 发起网络请求 |
| 剪贴板 | 读写剪贴板内容 |
| 通知 | 显示系统通知 |
| 进程 | 管理系统进程 |

### 权限请求

首次使用需要权限的插件时：
1. 显示权限请求对话框
2. 说明需要的权限
3. 用户选择授予或拒绝

### 管理权限

在插件详情中可以：
- 查看已授权限
- 撤销权限
- 重新授权

## 常见问题

### Q: 插件无法启用？
**A**:
1. 检查权限是否授予
2. 查看错误日志
3. 重启应用

### Q: 禁用后数据丢失？
**A**:
禁用插件不会删除数据，重新启用即可恢复。

### Q: 如何删除插件？
**A**:
内置插件无法删除，只能禁用。未来版本支持删除第三方插件。

### Q: 插件列表加载慢？
**A**:
1. 减少启用的插件数量
2. 清除缓存
3. 重启应用
