# 全局搜索

全局搜索是 LTools 的核心功能，让你快速访问所有插件和功能。

## 功能特性

- ✅ 快速唤起搜索窗口
- ✅ 模糊搜索匹配
- ✅ 插件、应用、文件路径统一搜索
- ✅ 键盘快捷键支持
- ✅ 实时结果展示

## 使用方法

### 打开搜索窗口

**快捷键**：
- macOS: `Cmd + 5`
- Windows/Linux: `Ctrl + 5`

或点击系统托盘图标选择「搜索」。

### 搜索内容

在搜索框中输入关键词，可以搜索：

**插件功能**：
- 输入插件名称或关键词
- 如：`calc`、`json`、`clipboard`

**应用程序**（应用启动器插件）：
- 输入应用名称
- 如：`chrome`、`vscode`、`wechat`

**文件路径**（如果启用）：
- 输入路径关键词
- 如：`/Users`、`Documents`

### 搜索结果

搜索结果显示：
- 匹配的项目列表
- 项目类型图标
- 简短描述
- 使用 `↑` `↓` 选择
- 按 `Enter` 打开

## 搜索技巧

### 模糊匹配

- 部分匹配即可
- 不区分大小写
- 支持拼音搜索

### 快捷操作

- `↑` / `↓`：选择结果
- `Enter`：打开选中项
- `Esc`：关闭搜索
- `Tab`：切换搜索类型

### 使用频率

常用项目会自动排在前面：
- 根据使用频率排序
- 最近使用优先显示
- 智能学习使用习惯

## 搜索范围

### 插件搜索

搜索所有已启用插件：
- 按名称搜索
- 按关键词搜索
- 按描述搜索

### 应用搜索

搜索已安装的应用程序：
- 自动扫描应用列表
- 缓存应用信息
- 支持快速刷新

### 路径搜索（可选）

搜索文件和目录：
- 需要启用路径搜索
- 显示文件图标
- 快速打开文件或目录

## 配置

### 快捷键设置

自定义搜索快捷键：
1. 打开「设置」→「快捷键」
2. 找到「全局搜索」
3. 设置新的快捷键

### 搜索设置

可配置选项：
- 默认搜索类型
- 结果数量限制
- 搜索历史保存

## 使用场景

### 快速访问插件

1. 按 `Cmd+5` 打开搜索
2. 输入 `json`
3. 回车打开 JSON 编辑器

### 启动应用

1. 打开搜索
2. 输入应用名
3. 回车启动

### 搜索书签

1. 打开搜索
2. 输入关键词
3. 选择书签并打开

- 插件启动时建立索引
- 增量更新索引
- 快速查询响应

### 缓存策略

- 缓存搜索结果
- 缓存应用列表
- 定期刷新缓存

## 常见问题

### Q: 搜索不到插件？
**A**:
1. 检查插件是否已启用
2. 刷新插件列表
3. 检查关键词是否正确

### Q: 搜索速度慢？
**A**:
1. 减少启用的插件数量
2. 清除搜索历史
3. 重启应用

### Q: 快捷键不工作？
**A**:
1. 检查快捷键冲突
2. macOS 检查辅助功能权限
3. 重新设置快捷键
